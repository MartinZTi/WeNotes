# 1. Collection概述
###### Collection  -- <font color=tomato>单列集合</font>
:::alert-light
<font color=gold>**Collection两个子接口：**</font>
* `java.util.List` 	*（元素有序、可重复）*
	* ArrayList
	* LinkedList
* `java.util.Set`  	*（元素无序，且不可重复）*
	* HashSet
	* TreeSet
:::
